Methodology

Data Source & Query

Data pulled from Org62 via SOQL on June 16, 2026. Q3 FY27 = Aug 1 – Oct 31, 2026.

SELECT Id, Name, Account.Name, Account.CSG_Region__c, Owner.Name, StageName, CloseDate,
       Targeted_Clouds__c, Forecasted_Attrition__c, Forecasted_Renewal_Rate__c,
       sfbase__ACV__c, Forecasted_Annual_Contract_Value__c,
       License_Renewal_Status__c, License_At_Risk_Reason__c, ACV_Reason_Detail__c,
       IsPremier_Attached__c, SEM_Notes__c
FROM Opportunity
WHERE CloseDate >= 2026-08-01 AND CloseDate <= 2026-10-31
  AND Targeted_Clouds__c INCLUDES ('Commerce Cloud','B2B Commerce','Order Management')
  AND Forecasted_Attrition__c <= -100000
  AND Account.CSG_Region__c = 'AMER PACE'
ORDER BY Forecasted_Attrition__c ASC

Note: No StageName filter applied. Q3 has not closed — all stages included. 13 records returned.
