Methodology

Data Source

Salesforce Org62 via SOQL query. Pull date: June 2026.

SELECT Id, Name, Account.Name, Account.CSG_Subregion__c, Account.CSG_Portfolio__c,
       Owner.Name, StageName, CloseDate, Forecasted_Attrition__c, Forecasted_Renewal_Rate__c,
       IsPremier_Attached__c, Targeted_Clouds__c, License_Renewal_Status__c,
       License_At_Risk_Reason__c, ACV_Reason_Detail__c, SEM_Notes__c,
       sfbase__ACV__c, Forecasted_Annual_Contract_Value__c
FROM Opportunity
WHERE CloseDate >= 2026-11-01 AND CloseDate <= 2027-01-31
  AND Targeted_Clouds__c INCLUDES ('MC ExactTarget','Marketing Cloud','Pardot','Datorama')
  AND Forecasted_Attrition__c <= -100000
  AND Account.CSG_Region__c = 'AMER PACE'
ORDER BY Forecasted_Attrition__c ASC

Note: Q4 is an open/pipeline quarter. StageName filter is omitted to capture all in-flight records regardless of stage.
