Methodology

Data Source & Filters

SOQL query against Org62 Opportunity object with the following filters:

  • Close Date: Nov 1, 2026 – Jan 31, 2027 (Q4 FY27)
  • Targeted Clouds: Core
  • Forecasted Attrition: ≤ -$100,000 (at-risk attrition ≥ $100K)
  • CSG Region: AMER PACE
  • Stage Filter: None (all stages included — quarter has not closed)
  • Sort: Forecasted Attrition ascending (highest attrition first)
  • Result Count: 100 records
SELECT Id, Name, Account.Name, Account.CSG_Subregion__c, Account.CSG_Portfolio__c,
       Owner.Name, StageName, CloseDate, Forecasted_Attrition__c,
       Forecasted_Renewal_Rate__c, IsPremier_Attached__c, Targeted_Clouds__c,
       License_Renewal_Status__c, License_At_Risk_Reason__c, ACV_Reason_Detail__c,
       SEM_Notes__c, sfbase__ACV__c, Forecasted_Annual_Contract_Value__c
FROM Opportunity
WHERE CloseDate >= 2026-11-01 AND CloseDate <= 2027-01-31
  AND Targeted_Clouds__c INCLUDES ('Core')
  AND Forecasted_Attrition__c <= -100000
  AND Account.CSG_Region__c = 'AMER PACE'
ORDER BY Forecasted_Attrition__c ASC

Classification Logic (Projected)

  • Full vs Partial: License_Renewal_Status__c = "Will Attrit" (exact) = Full; anything else = Partial
  • Attrition Type: Opp Name contains "Off Cycle" / "Offcycle" / "Non-Coterminous" = Offcycle; otherwise = Oncycle
  • Not Actionable: Rate=100 + "Will Renew (Flat)"; Notes say "nonactionable"; #LicNoAttrit + rate=100 with no risk signals; Empty notes + no status + closed stage
  • Logo Loss (Projected): Rate=0 or None + "Will Attrit" (exact, not Actionable) + no active plays; Dead Attrition stage; Empty notes + no status + no rate
  • Partial Save (Projected): Rate between 0-100; Rate=100 + "Will Reduce"; #LicPartialAttrit present; Status contains "Reduce" or "Actionable"; Active defense plays documented
  • SP Downgrade: #LicNoAttrit + #SigRisk + "Will Reduce" in status (sig-only risk, license retained)

Note: All classifications are projected. Q4 FY27 (Nov 2026–Jan 2027) has not closed. Final outcomes will be determined after quarter close.

Area Mapping

  • RCG: Account.CSG_Subregion__c contains "AMER PACE RCG" (blue #0176d3)
  • MAE: Account.CSG_Subregion__c contains "AMER PACE MAE ENTR" (purple #7e57c2)
  • CMRCL: Account.CSG_Subregion__c contains "AMER PACE CMRCL" (orange #fe9339)

CSG Notes Compliance Rules (PACE Guidance)

  • License Tag (Required): Must include one of: #LicNoAttrit, #LicPartialAttrit, #LicFullAttrit
  • Sig Tag (Required if Premier=Yes): Must include one of: #SigRisk, #NoSigRisk
  • Date Stamp (Required): Must include a date in the note entry
  • Invalid Tags Flagged: #PartialAttrit, #FullAttrit, #NoAttrit, #fullattrit, #nonactionable, #partialattritrisk, #FullAttritRisk, #Attrit, #SigAttrit
  • Empty Notes: Automatic FAIL if CSG Notes field is empty
