Methodology

Data Source & Filters

SOQL query against Org62 Opportunity object with the following filters:

  • Close Date: Aug 1, 2026 – Oct 31, 2026 (Q3 FY27)
  • Targeted Clouds: MC ExactTarget, Marketing Cloud, Pardot, Datorama
  • Forecasted Attrition: ≤ -$100,000 (at-risk attrition ≥ $100K)
  • CSG Region: AMER PACE
  • Stage Filter: None (all stages included — quarter has not closed)
  • Sort: Forecasted Attrition ascending (highest attrition first)

Classification Logic (Projected)

  • Full vs Partial: License_Renewal_Status__c = "Will Attrit" (exact) = Full; anything else = Partial
  • Attrition Type: Opp Name contains "Off Cycle" / "Offcycle" / "Non-Coterminous" = Offcycle; otherwise = Oncycle
  • Not Actionable: Confirmed competitor win (signed/selected), M&A-driven, or offcycle structural change with no remaining swing
  • Logo Loss (Projected): Will Attrit + full platform exit indicated + no active plays documented
  • Partial Save (Projected): Status contains "Reduce" or "Actionable", or evidence of partial retention / active defense plays

Note: Classifications are projected. Final outcomes will be determined after quarter close.

CSG Notes Compliance Rules (PACE Guidance)

  • License Tag (Required): Must include one of: #LicNoAttrit, #LicPartialAttrit, #LicFullAttrit
  • Sig Tag (Required if Premier=Yes): Must include one of: #SigRisk, #NoSigRisk
  • Date Stamp (Required): Must include a date in the note entry
  • Invalid Tags Flagged: #PartAttrit, #FullAttrit, #NoAttrit, #fullattrit, #nonactionable, #PartialAttrition, #FullAttritRisk, #SigAttrit, #Attrit
  • Offense Tags in Wrong Location: #ARI, #SWAP, #RFP, #SIGTRIAL, #RA belong on Red Account ECOMM Headline, not in CSG Notes
