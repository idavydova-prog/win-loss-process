Methodology

Data Source & Filters

SOQL query against Org62 Opportunity object with the following filters:

  • Close Date: Feb 1, 2026 – Apr 30, 2026 (Q1 FY27)
  • Targeted Clouds: MC ExactTarget, Marketing Cloud, Pardot, Datorama
  • Forecasted Attrition: ≤ -$100,000 (attrition ≥ $100K)
  • CSG Region: AMER PACE
  • Stage: 05 Closed + Dead Attrition (quarter is closed)
  • Sort: Forecasted Attrition ascending (highest attrition first)

Classification Logic

  • Full vs Partial: License_Renewal_Status__c = "Will Attrit" (exact) = Full; anything else = Partial
  • Attrition Type: Opp Name contains "Off Cycle" / "Offcycle" / "Non-Coterminous" = Offcycle; otherwise = Oncycle
  • Not Actionable: Off-cycle forecasting records, bad debt, or DC credit assessments with no actionable swing
  • Win: High renewal rate (>89%) + documented CSM engagement + no full attrition risk
  • Logo Loss: Will Attrit / Dead Attrition + 0% rate + full platform exit confirmed
  • Partial Save: Will Reduce / Actionable with partial retention
  • SP Downgrade: Pardot license attriting while retaining other MC products

CSG Notes Compliance Rules (PACE Guidance)

  • License Tag (Required): Must include one of: #LicNoAttrit, #LicPartialAttrit, #LicFullAttrit
  • Sig Tag (Required if Premier=Yes): Must include one of: #SigRisk, #NoSigRisk
  • Date Stamp (Required): Must include a date in the note entry
  • Invalid Tags Flagged: #PartAttrit, #FullAttrit, #NoAttrit, #fullattrit, #nonactionable, #partialattrit, #NoSigAttrit, #LicPartrialAttrit
  • Offense Tags in Wrong Location: #ARI, #SWAP, #RFP, #SIGTRIAL, #SWE belong on Red Account ECOMM Headline, not in CSG Notes
